---
name: No AI mentions
description: Never mention AI in user-facing copy — the user's audience dislikes it
type: feedback
---

Never use the word "AI" or reference automation/machine learning in any user-facing copy, descriptions, or meta text on the site.

**Why:** The user says "people hate AI" — mentioning it undermines trust and turns readers off.

**How to apply:** When writing site copy, meta descriptions, footer text, about pages, or any public-facing content, describe the site as aggregating/summarizing news without referencing the technology behind it.
