Never use em dashes (—) in any writing. Use periods, commas, or parentheses instead.

Also avoid these AI writing tells: "delves", "tapestry", "vibrant", "bustling", "nestled", "landscape", "in conclusion", "it's worth noting", "importantly".

**Why:** The user says "NO EM DASHES or AI style writing BS". These are dead giveaways that content was generated.

**How to apply:** Applies to all user-facing copy, Claude API prompts, sample data, site copy, and summaries. Use short punchy sentences. Write like a real person texting their friend about neighborhood news.
